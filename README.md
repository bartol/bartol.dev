## Run locally

1. Run `cp .env.example .env` and add environment variables.

2. Install dependencies.

   ```bash
   yarn
   ```

3. Start development environment.

   ```bash
   yarn dev
   ```

4. Open browser and navigate to `localhost:8000`

5. Build project

   ```bash
   yarn build
   ```
