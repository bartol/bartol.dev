# vim

## Start vim without .vimrc

    $ vim -u NONE

## vimdiff save merge and exit

    :w
    :qa

## open terminal in vertical split

	:vert term

## open terminal in horizontal split

	:term

## switch to last used buffer

	:b#

## save file

	:w

## save file and exit

	:wq

## exit without saving

	:q!
