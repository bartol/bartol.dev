# Zsh keyboard shortcuts

- `^A` - beginning of the line
- `^E` - end of the line
- `alt + b` - move one word backward
- `alt + f` - move one word forward
- `^U` - delete whole line
- `^K` - delete after cursor
- `^W` - delete word before cursor
- `alt + d` - delete word after cursor
- `^R` - search history
- `^G` - escape from search
- `^_` - undo last change
- `^L` - clear screen
- `^C` - kill process
- `^Z` - suspend process
